# TruPath — Master System Overview

**Last Updated:** 2026-03-24

---

## 1. Company Definition

**TruPath** is a coaching and transformation company built on a proprietary behavioral pattern engine that identifies, predicts, and reshapes human behavior across multiple life domains.

### Delivery Model

TruPath delivers its work through:
- 1:1 coaching
- Small group coaching
- Mid-size group programs
- Live in-person workshops
- Live virtual courses

The coaching experience is supported and enhanced by a software system that enables scalable assessment, analysis, and personalized outputs.

---

## 2. The Core Innovation

Most systems fall into one of three categories:

- **Advisory systems** — tell people what to do
- **Assessment systems** — snapshot who someone is
- **Coaching systems** — guide behavior through interaction

TruPath unifies all three into a continuous behavioral engine:

> **Every interaction both guides behavior and updates a structured model of that behavior.**

**Input → Pattern Detection → Intervention → Outcome → Model Update**

This creates a system that not only helps people change — but learns how change happens and improves its ability to drive it.

## 3. Core Intellectual Property (Behavioral Pattern Engine)

At the center of TruPath is a **behavioral pattern engine** — a multi-domain human behavior system that identifies, predicts, and reshapes outcomes by diagnosing underlying behavioral patterns that operate consistently across all areas of life.

### 3.1 Three Domains of Disconnection

1. **Disconnection from Self** (low self-worth)
2. **Disconnection from Others** (conditional love)
3. **Disconnection from All That's Greater** (lack of trust / fear / control)

### 3.2 The Six Core Patterns

Each domain contains one **active** pattern (doing/creating) and one **passive** pattern (allowing/withdrawing):

| Domain | Active Pattern | Passive Pattern |
|--------|---------------|-----------------|
| Self | False Self-View | External Validation |
| Others | Showing Love | Receiving Love |
| Greater | Control | Fear |

### 3.3 Pattern Structure

Each pattern is defined across four components:
1. **Core Belief / Untruth**
2. **Observable Behaviors**
3. **Emotional States**
4. **Measurable Consequences**

### 3.4 Reinforcing Loop Mechanism

Each pattern operates as a closed loop:

**Belief → Behavior → Outcome → Reinforces Belief**

These loops are self-sustaining, produce predictable outcomes, and can be interrupted through targeted intervention.

### 3.5 Multi-Pattern System

Individuals typically exhibit:
- One **primary pattern**
- One or more **secondary patterns**

The system includes co-occurrence modeling, reinforcement relationships between patterns, and pattern interaction logic — enabling more accurate diagnosis, deeper behavioral insight, and better intervention targeting.

## 4. Multi-Domain Model

TruPath applies the same pattern engine across four major life domains:

1. **Finance**
2. **Relationships** (personal and professional)
3. **Health**
4. **Purpose** (career, job, calling)

**Key principle:** The pattern is primary. The domain is the expression. The same underlying pattern appears in all domains, drives consistent behavior, and produces similar outcomes in different contexts.

---

## 5. Entry Point Model ("4 Doors")

Each domain acts as an independent entry point into the system.

**Process Flow:**
1. A user enters through a domain where they feel pain (e.g., finance)
2. The system identifies their underlying pattern(s)
3. The same pattern is mapped across all other domains
4. The user gains domain-specific insight and cross-domain awareness

- Financial behaviors (spending, saving, debt patterns)
- Decision-making tendencies
- Risk behaviors
- Relationship dynamics
- Environmental outcomes

## 8. The Dual-Sided System

### Side A — Direct Human Impact (Coaching & Transformation)

Delivered through coaching, structured programs, and software tools:
- Diagnoses behavioral patterns
- Guides decisions and actions
- Produces measurable personal change

### Side B — Behavioral Intelligence & Influence Layer

Simultaneously, TruPath produces a continuously updated, structured model of human behavior tied to real-world actions and outcomes. This enables:

1. **Real-Time Behavior Influence** — Live interaction analysis, pattern identification, guided decisions
2. **System-Level Behavior Optimization** — Identifying root causes of action failure, adjusting flows and decision pathways
3. **Agent & Decision System Integration** — Embedding behavioral models into AI systems for dynamic adaptation

### The Bridge Between Sides

Side A and Side B reinforce each other:
- Coaching interactions generate behavioral data
- Data refines the model
- Model improves influence
- Influence improves outcomes

> The system improves as it is used.

TruPath builds a persistent behavioral model of each individual through:
- Structured assessments (tools)

## 11. Financial TruPath v3 (Software System)

Financial TruPath v3 is the primary implementation of the TruPath engine in the financial domain.

- Built on Google Apps Script
- Uses Google Sheets as a database
- Integrates GPT-4o for narrative analysis
- Generates dashboards and PDF reports

**Scale:** ~69,000 lines of production JavaScript across 89 files, plus 11,700 lines of HTML templates. Eight interconnected tools, a cross-tool insights engine, a capstone integration layer, and a full admin system.

### 8-Tool System Flow

```
PSYCHOLOGICAL FOUNDATION          FINANCIAL APPLICATION
========================          =====================
Tool 1: Core Pattern        →    Tool 2: Financial Clarity
Tool 3: Identity Grounding  →    Tool 4: Budget Framework
Tool 5: Love Grounding      →    Tool 6: Retirement Blueprint
Tool 7: Control/Fear Ground →    Tool 8: Investment Planning

                    ↓
            Capstone AI Integration
```

### AI System
- GPT used for narrative insight (GPT-4o and GPT-4o-mini)
- 3-layer fallback system ensures 100% reliability
- Cross-tool insights engine adapts downstream tools
- Capstone synthesizes full user profile

### Core Software Architecture
- Router, Tool Registry, Framework Core
- Data Service, Response Manager
- Insights Pipeline, Authentication
- Spreadsheet Cache
- Data stored in Google Sheets (14-tab system)

## What Makes TruPath Unique

TruPath is not a coaching program, a static assessment, a financial tool, or a chatbot.

It is a system that continuously models human behavior and uses that model to actively influence outcomes — and it improves at both over time.

Most systems either help people change **or** analyze behavior. TruPath does both simultaneously.

This represents a shift from:
- Static advice → dynamic intelligence
- Episodic coaching → continuous modeling
- Individual insight → system-level application

Positioning it as a foundational platform for human development, decision intelligence, and behavior-driven systems across industries.
