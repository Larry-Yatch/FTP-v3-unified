# TruPath — Master System Overview

## 1. Company Definition

TruPath is a coaching and transformation company built on a proprietary behavioral pattern engine.

It delivers transformation through:
- 1:1 coaching
- Small and mid-size group programs
- Live and virtual courses
- Workshops

The company is supported by a software system that enables scalable assessment, analysis, and personalized outputs.

---

## 2. Core System Definition

TruPath is:

A multi-domain human behavior engine that identifies, predicts, and reshapes outcomes by diagnosing underlying behavioral patterns that operate consistently across all areas of life.

## 7. Financial TruPath v3 (Software System)

Financial TruPath v3 is the primary implementation of the TruPath engine in the financial domain.

It is:
- Built on Google Apps Script
- Uses Google Sheets as a database
- Integrates GPT-4o for narrative analysis
- Generates dashboards and PDF reports

Scale:
- ~69,000 lines of JavaScript
- 8-tool system
- Full admin + client system
