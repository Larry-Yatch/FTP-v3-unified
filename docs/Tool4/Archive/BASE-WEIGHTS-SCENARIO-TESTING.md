# Tool 4: Base Weights Scenario Testing

**Date:** November 18, 2025
**Purpose:** Validate M/E/F/J base weights with 20 realistic student scenarios
**Status:** üîÑ In Progress

---

## Testing Methodology

For each scenario:
1. Define student financial situation
2. Apply base weights from their selected priority
3. Calculate dollar amounts (not just percentages)
4. Evaluate: "Does this feel right for this student?"
5. Identify any unrealistic allocations

**Target Coverage:**
- Low income: $2K-$3.5K/month (6 scenarios)
- Medium income: $4K-$7K/month (8 scenarios)
- High income: $8K-$15K+/month (6 scenarios)
- Various debt levels (none, moderate, high)
- Various emergency funds (none, 1mo, 3mo, 6mo+)
- All 10 priorities tested

## SUMMARY OF FINDINGS

### üî¥ **Critical Issues Identified:**

#### **Issue 1: Low-Income + High-E Priorities = Unrealistic Gaps**

**Problem Scenarios:**
- Scenario 1: "Stabilize to Survive" (50% E) with $2,800 income
- Scenario 2: "Reclaim Control" (35% E) with $3,200 income
